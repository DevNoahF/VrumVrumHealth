Sobre Nós

Este é um trabalho desenvolvimento pelos estudantes de Ciência da Computação da Unimar com o objetivo de trazer uma solução inteligente e fácil do problema que acerca a realização de agendamentos para, exames, consultas e tratamentos contínuo. O projeto foi desenvolvido em Java utilizando SpringBoot e SpringBoot Security com frameworks principais, h2 e docker para banco de dados, e, por fim, foi utilizado HTML,CSS E JavaScript para desenvolvimento de uma interface Gráfica de fácil reconhecimento

O que é o Vrum Vrum Health

A Vrum Vrum Health é uma solução moderna criada para ajudar pessoas que necessitam de transporte para consultas, exames e atendimentos médicos. Nosso sistema permite que os usuários agendem viagens de forma simples, recebam confirmações imediatas e acompanhem o motorista em tempo real. Garantimos praticidade, segurança e acessibilidade no cuidado com sua saúde. Nosso objetivo é tornar o processo de deslocamento para hospitais mais confortável, humano e eficiente.
